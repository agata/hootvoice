HootVoice

音声で文字入力。
プライバシーを守る設計。

Whisper による高速かつプライベートな音声入力。小さなフローティングツールバー、デバイス制御、ユーザー辞書を備えています。すべてローカルで完結し、APIキー不要、無料で使えます。

Windows (x86_64) macOS (arm64) Linux AppImage (x86_64)

なぜ HootVoice？

驚くほど高速

最適化された Whisper の文字起こしは、あなたの思考の速さに追いつきます。

プライベートかつローカル

すべてがあなたのマシン上で完結し、データが外部に送られることはありません。

手間いらずのワークフロー

グローバルホットキー、自動コピー&ペースト、カスタム辞書、音量調整可能なステータスサウンド。

はじめに

  • 使い方のヒントは ユーザーマニュアル をご覧ください

動作条件

  • macOS: macOS 12 以降を実行する Apple シリコン搭載 Mac
  • Windows: Windows 10/11 64ビット PC（最新の GPU ドライバー推奨）
  • Linux: Vulkan ランタイム/ドライバー、glslc、libopenblas — 詳しくは Linux セットアップ
